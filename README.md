# מסע יציאת מצרים: אל החופש! 🐪
גרסת דפדפן (HTML/JavaScript) — בלי pygbag, בלי WASM, בלי כאבי ראש

---

## מה זה הקובץ הזה
`index.html` הוא **המשחק כולו** — קוד, גרפיקה, לוגיקה, הכל בקובץ אחד.
אין צורך בקבצים נוספים, אין צורך בפונט חיצוני (העברית מוצגת
באמצעות הפונטים המובנים של הדפדפן, שתמיד תומכים בעברית), ואין
צורך בשום שרת מקומי או תהליך build.

## איך לבדוק שזה עובד (לפני העלאה)
פשוט תלחצי פעמיים על הקובץ — הוא ייפתח בדפדפן ברירת המחדל
ויעבוד מיידית. אין שום שלב נוסף.

## איך לפרסם בקישור (GitHub Pages)

1. ב-GitHub, צרי ריפו חדש (או השתמשי בקיים)
2. העלי את `index.html` **לשורש** הריפו (לא בתוך תיקיית משנה)
3. Settings → Pages → Source: Deploy from a branch → main → / (root) → Save
4. תוך דקה-שתיים יופיע קישור כמו:
   `https://[שם-המשתמש].github.io/[שם-הריפו]/`

זהו. אין build, אין cache, אין קבצי CDN חיצוניים שיכולים
להיכשל — רק קובץ HTML אחד שהדפדפן קורא ומריץ ישירות.

## בקרות
- **חצים / WASD** — תזוזה ימינה/שמאלה
- **רווח / חץ למעלה / W** — קפיצה
- **לחיצה על המסך** — מתחילה את המשחק / ממשיכה אחרי הודעה

## תמיכה במובייל/מסך מגע
המשחק מציג כפתורי מגע (◀ ▶ ⤴) בתחתית המסך באופן אוטומטי
במכשירים עם מסך צר (טאבלטים, טלפונים). במחשבים עם מסך רגיל,
הכפתורים לא מוצגים — רק המקלדת.

## אם תרצי לשנות משהו בעתיד
כל הקוד (כולל הגרפיקה, רמות המשחק, והטקסטים בעברית) נמצא
בתוך תגי `<script>` בקובץ `index.html` עצמו. זה קובץ טקסט רגיל —
ניתן לפתוח אותו בכל עורך טקסט (Notepad, VS Code) ולערוך ישירות.
